# 프로그래밍 핵심 개념 MOC

## 개요
이 MOC는 프로그래밍의 기본이 되는 핵심 개념들을 다룹니다. 각 개념은 언어에 독립적이며, 특정 프로그래밍 언어를 학습할 때 기반이 되는 지식들입니다.
